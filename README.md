# supplement2
A c++ code that calculates the area of a rectangle, circle, and triangle.

# How to run
1) make a /build dir
```
mkdir build/
```
2) change directory inside the build dir
```
cd build/
```
3) run cmake and make
```
cmake .. && make
```
4) run ctest on the file or just run the executable

- ctest:
```
ctest
```
- executable:
```
./shape_test
```
